## What's New

This release introduces a comprehensive admin feedback system and documentation improvements.

### Features

- **GridBackground component** for customizable visual design
- **Admin feedback system** with filters, tables, and detailed views
- **Documentation system** with dynamic content and sidebar navigation
- **Last used provider tracking** for enhanced sign-in experience
- **Contact and about pages**

### Maintenance

- Update Next.js to 15.5.0
- Update React to 19.1.1
- Update AI SDK to v5
- Update all Radix UI component packages
- Update testing dependencies and build tools
- Claude Code integration with new agents, commands, and formatting hooks
- Improve API file organization and documentation structure
